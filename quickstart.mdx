# Quickstart

## Prerequisites

- SSH access to `109.199.120.120`
- Docker and Docker Compose installed
- Root or sudo access

## Start the Full Stack

```bash
cd /opt/coderz
docker compose up -d
```

## Verify Everything is Running

```bash
docker compose ps
```

You should see all containers with status `running` or `healthy`.

## Common Operations

<AccordionGroup>
  <Accordion title="Restart a single service">
    ```bash
    cd /opt/coderz
    docker compose restart grafana
    ```
  </Accordion>

  <Accordion title="View logs of a service">
    ```bash
    docker compose logs -f grafana
    docker compose logs -f coderz-dotnet-api --tail=100
    ```
  </Accordion>

  <Accordion title="Stop the entire stack">
    ```bash
    cd /opt/coderz
    docker compose down
    ```
  </Accordion>

  <Accordion title="Update a service image">
    ```bash
    cd /opt/coderz
    docker compose pull grafana
    docker compose up -d grafana
    ```
  </Accordion>

  <Accordion title="Check disk usage">
    ```bash
    docker system df
    df -h
    ```
  </Accordion>
</AccordionGroup>

## Mintlify Docs Server

The docs server runs as a systemd service:

```bash
systemctl status mintlify
systemctl restart mintlify
journalctl -u mintlify -f
```

Docs files are in `/opt/mintlify-docs/`.
